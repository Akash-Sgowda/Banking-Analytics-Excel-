# Banking-Analytics-Excel-

# Project Overview
The Bank Analytics Dashboard is an end-to-end data analytics project designed to analyze banking loan data and deliver actionable insights for decision-making.
This project focuses on understanding loan distribution, customer behavior, risk patterns, and financial performance using interactive dashboards.
The dashboards provide a clear overview of key banking metrics, enabling stakeholders to monitor trends, evaluate loan quality, and identify high-risk and high-value segments efficiently.

# Objectives
Analyze customer loan data to identify trends and patterns.  
Understand loan purposes and customer borrowing behavior.  
Evaluate loan performance based on grades, verification status, and ownership.  
Provide insights that support risk assessment and strategic decisions.  

# Overview Dashboard
This dashboard provides a high-level summary of banking operations:   
Total members and accounts overview.  
Average interest rate and loan amount.  
Distribution of loans by purpose.  
Loan status analysis (Current, Fully Paid, Charged Off).  
State-wise loan status comparison.  
Grade-wise loan amount and interest rate analysis.   

# Finance Dashboard
This dashboard focuses on financial and risk-related metrics:  
Maximum, minimum, median, and average loan amounts.  
Subgrade-wise revolving balance analysis.  
Average Debt-to-Income (DTI) ratio by loan purpose.  
Monthly loan status trends.  
Loan amount sanctioned based on home ownership.    
Count of customers taking loans for different purposes.  


# Key Insights
Debt consolidation and credit card loans are the most common loan purposes.  
Customers with verified income tend to show better loan performance.  
Higher loan grades are associated with lower interest rates.  
Home ownership has a significant impact on loan sanction amounts.  
Loan activity shows seasonal trends across months.  

# Business Value
Helps banks understand customer borrowing patterns.  
Supports better credit risk assessment.  
Assists management in identifying profitable and risky loan segments.  
Enables data-driven decision-making through visual insights.
